# Synthetic Photometry Basic Dataset

This is a small bundled synthetic RWD-style dataset for the GUI quickstart.

Purpose:
- demonstrate the software workflow end to end
- provide a committed dataset that users can run without generating data first
- exercise signal/reference pairing, tonic trends, phasic events, and standard output generation

Contents:
- two RWD-style session folders
- each session contains `fluorescence.csv`
- two ROIs/channels: `CH1` and `CH2`
- `tutorial_config.yaml` with settings used by the quickstart
- `generation_manifest.yaml` recording the generator command and synthetic-data metadata

This dataset is synthetic. It is for workflow demonstration and software testing, not biological validation.
